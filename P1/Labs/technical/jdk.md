# JDK (Java Development Kit)

The JDK is what developers install to build Java programs.

It contains everything:

JDK = JRE + development tools

| Tool      | Purpose                |
| --------- | ---------------------- |
| `javac`   | Java compiler          |
| `java`    | Run Java program       |
| `jar`     | Package programs       |
| `javadoc` | Generate documentation |
| `jdb`     | Java debugger          |
