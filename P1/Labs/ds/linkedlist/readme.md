# What is a Linked List?

A Linked List is a data structure where elements are stored in nodes, and each node points to the next node.

Head
 ↓
[10 | •] → [20 | •] → [30 | •] → [40 | null]



    data = the value

    next = reference to the next node

| Feature | Array      | Linked List |
| ------- | ---------- | ----------- |
| Memory  | contiguous | scattered   |
| Size    | fixed      | dynamic     |
| Insert  | slow       | fast        |
| Access  | fast       | slower      |
